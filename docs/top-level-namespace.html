#Bool(arg) ⇒ Boolean

Converts arg to a boolean (true or false).

Returns the truthiness of obj, as either true or false. This is functionally equivalent to calling !!arg.

Examples:

require 'mug/bool'
Bool(nil)    #=> false
Bool(0)      #=> true
Bool("hi")   #=> true
